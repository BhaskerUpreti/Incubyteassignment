# Gmail Compose Function – Manual Testing Assessment

## Overview

This repository contains manual test cases for the Gmail Compose functionality.

The assessment covers:

* Sending an email with:

  * Subject: "Incubyte"
  * Body: "QA test for Incubyte"
* Traditional Style Test Cases (  Positive Test Scenarios, Negative Test Scenarios )
* BDD (Behavior-Driven Development) Style Test Cases
